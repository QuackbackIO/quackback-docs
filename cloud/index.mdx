---
title: Managed Cloud
description: Quackback Cloud is the hosted, managed version of Quackback. Self-hosting is also fully supported via the open-source bundle.
---

Quackback comes in two editions:

- **Quackback OSS** — fully open-source under AGPL. Run it yourself on any
  Docker-capable infrastructure. Every feature is included with no limits.
- **Quackback Cloud** — managed hosting on our infrastructure with three
  tiers: Starter ($19/mo), Pro ($79/mo), and Scale ($299/mo). Tier limits
  apply only to Cloud — see [Tiers](./tiers) for the full matrix.

The product code is the same. The Cloud edition adds:

- A managed control plane (provisioning, billing, upgrades)
- Per-tenant isolation on shared Postgres clusters
- Per-tenant Let's Encrypt certificates and a default subdomain
- Custom-domain support on Pro and above
