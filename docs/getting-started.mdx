# Getting Started

This guide will help you set up Quackback and start collecting feedback from your users.

## Prerequisites

Before you begin, make sure you have:

- A Quackback account
- Access to your application's codebase

## Create a Workspace

1. Sign in to your Quackback account
2. Click "Create Workspace"
3. Enter your workspace name and details
4. Choose your plan

## Install the Widget

Add the Quackback widget to your application:

```typescript
import { QuackbackWidget } from '@quackback/widget'

function App() {
  return (
    <QuackbackWidget
      workspaceId="your-workspace-id"
      theme="dark"
    />
  )
}
```

## Configure Your Board

Customize your feedback board:

- Set up categories for different types of feedback
- Configure voting settings
- Enable or disable anonymous submissions

<Callout type="info" title="Pro Tip">
Start with a simple setup and add complexity as you learn what works for your users.
</Callout>
