# Single sign-on

SSO via OpenID Connect (OIDC) or SAML is not currently implemented in Quackback. This feature may be added in a future release.

## Current Authentication Options

Quackback supports the following passwordless authentication methods:

| Method | Portal Users | Team Members |
|--------|--------------|--------------|
| Email OTP | Yes | Yes |
| GitHub OAuth | Yes | Yes |
| Google OAuth | Yes | Yes |
